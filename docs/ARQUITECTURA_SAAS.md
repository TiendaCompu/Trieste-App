# ZYTRONIX - Arquitectura SaaS

## Identidad

ZYTRONIX significa Sistema Electronico Automatizado.

El producto inicial sera una plataforma de gestion para talleres automotrices.

## Concepto principal

ZYTRONIX es la plataforma.

Cada taller es un cliente dentro de la plataforma.

Ejemplo:

- Plataforma: ZYTRONIX
- Cliente piloto: Centro de Servicio Automotriz Trieste

## Modelo multiempresa

Cada taller tendra datos separados:

- usuarios
- clientes
- vehiculos
- ordenes de trabajo
- presupuestos
- notas de entrega
- inventario
- reportes
- configuracion

## Stack oficial

Frontend:

- React
- Tailwind CSS

API:

- FastAPI
- Python

Base de datos:

- PostgreSQL

Infraestructura:

- Docker
