# Trieste App - Documento Inicial

## Estado del proyecto

Proyecto en fase de reconstrucción profesional.

Actualmente se está trabajando en:

- arquitectura del sistema
- experiencia visual
- flujo operativo de taller
- modelo SQL
- prototipo interactivo

## Objetivo principal

Construir un sistema moderno de gestión de taller automotriz para uso local y multiusuario dentro de una red LAN.

## Características clave

- búsqueda rápida por matrícula
- histórico completo del vehículo
- presupuestos profesionales
- control de repuestos
- control de órdenes de trabajo
- soporte multi moneda
- control cambiario
- mecánicos y comisiones
- diseño moderno claro/oscuro

## Estado actual

Se está diseñando primero el flujo visual y operativo antes de iniciar la programación definitiva.
