# Trieste App

Sistema de gestión operativa de taller automotriz.

## Objetivo

Trieste App es un sistema diseñado para controlar el flujo completo de trabajo de un taller automotriz:

- ingreso de vehículos
- diagnóstico
- presupuestos
- repuestos
- compras
- reparación
- histórico
- notas de entrega
- control operativo

## Filosofía del proyecto

Este NO es un sistema contable completo.

El enfoque principal es la gestión real del taller y la operación diaria.

## Tecnologías previstas

### Frontend
- React
- Tailwind CSS
- Interfaz moderna responsive
- Modo claro / oscuro

### Backend
- Python FastAPI
- SQLAlchemy

### Base de datos
- PostgreSQL local
