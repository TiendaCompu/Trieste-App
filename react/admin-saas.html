Decisiones SaaS importantes

La plataforma debe guardar cada taller como un tenant separado. Cada usuario, vehículo, orden, inventario y cliente debe pertenecer a un taller específico.

Esto permitirá vender ZYTRONIX como suscripción sin mezclar datos entre clientes.
